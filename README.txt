## Robot Test Cases for Booking.com Website

### Test Cases

TS 1: Search For Hotel
   - Open browser and go to https://www.booking.com/
   - Enter destination (e.g., Almaty)
   - Select check-in and check-out dates
   - Click "Search"
   - Verify search results are displayed

TS 2: Filter By 5-Star Hotels
   - Open browser and go to https://www.booking.com/
   - Enter destination (e.g., Almaty)
   - Click "Search"
   - Apply filter for 5-star hotels
   - Verify that only 5-star hotels are shown

TS 3: Verify Search Results Contain Destination
   - Open browser and go to https://www.booking.com/
   - Search for a specific city (e.g., Almaty)
   - Verify that the city name appears in the search results

TS 4: Change Website Language
   - Open browser and go to https://www.booking.com/
   - Click on the language icon in the header
   - Select a different language (e.g., English)
   - Verify that the page language changes successfully

TS 5: Change Currency
   - Open browser and go to https://www.booking.com/
   - Click on the currency icon in the header
   - Select another currency (e.g., USD)
   - Verify that the currency displayed on hotel prices changes
